# TLS (Transport layer security)

La TLS (_transport layer security_ o seguridad de la capa de transporte) es un protocolo de seguridad definido por la IETF (Internet Engineering Task Force) que se basa en reemplazar SSL 3.0 (Secure Sockets Layer 3.0).
